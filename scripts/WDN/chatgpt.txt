- Disc types (mass, radius, name)
- Reactions
 - Unimolecular reactions A -> B and A -> B + C with probability per second
 - Bimolecular reactions A + B -> C and A + B -> C + D with probability per collision
- Simulation radius
- Disc count of the simulation (discs are distributed randomly across a uniform grid)
- Initial disc type distribution (like A: 0.3, B: 0.7, C: 0.0)
- Most probable speed of the discs
