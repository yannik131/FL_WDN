\section{Methodology}
The main challenge in this project is that the model should support an arbitrary number of disc types and reactions. A fixed-size MLP, for example, would fail for two reasons:

\begin{itemize}
    \item With a fixed number of input nodes, any input tuple $(\Delta t, v_{\text{mp}}, D, \Omega, R, N, f)$ would have to be compressed or padded into the same input vector size. If the number of input parameters exceeds the number of input nodes, information loss is inevitable
    \item Re-ordering of reactions or disc types in the input vector would result in a different representation. A simulation with disc types $D = \{A, B\}$ would look entirely different from $D = \{B, A\}$ to the model
\end{itemize}

A graph neural network solves both of these issues. They guarantee that permutations of the same input graph result in the same internal representation. An example representation of a system with three species $D = \{A, B, C\}$ and two reactions $\Omega = \{\{A+B\rightarrow C, p_1\}, \{C \rightarrow A+B, p_2\}$ is shown in \cref{fig_reaction_graph_example}.
